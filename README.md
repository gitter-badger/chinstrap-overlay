# Chinstrap overlay
> Over the edge

This overlay provides a Gentoo based install with too much unmasked versions.

![](header.png)

## Installation

Gentoo:

```sh
eselect repository add chinstrap git https://git.gomersbach.nl/mgomersbach/chinstrap.git
```


## Usage example

emerge -uDNg @world

## Development setup

```sh
emerge git eselect-repository
```
